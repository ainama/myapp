# myapp

frog社区

后端使用：node.js + express + mysql
前端使用：react + react-router + redux
使用插件：simditor


# use

```
  git clone https://github.com/ainama/myapp.git

  cd myapp

  npm install

  run start

  浏览器输入地址：http://localhost:3000/
```
