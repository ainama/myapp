# use

```
  git clone https://github.com/ainama/myapp.git

  cd myapp

  npm install

  run start

  // 出于对数据库信息的保护，暂不能以此方式访问，如为公司内部人员，请联系小组成员索要数据库信息
  浏览器输入地址：http://localhost:3000/
```
